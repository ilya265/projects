всё что находится в папке htdocs является главной частью проекта.

В файле settings.php содержатся переменные для доступа к БД
файл обязательно должен находиться на 1 папку выше основного
проекта.

В файле create_database.php содержатся запросы для создания
базы данных а так же запрос для заполнения таблицы парой
тестовых форм.
